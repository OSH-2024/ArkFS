## 关于项目

大语言模型在近几年的发展中取得了巨大的突破。目前已有许多研究人员将大模型应用于操作系统，如则是最近发表的大语言模型在操作系统层级应用的重要成果。文件系统是操作系统的重要组成部分，也有许多在探索其优化方向以及改进方法。本小组调研了三种大语言模型以及文件系统目前存在的部分问题。
